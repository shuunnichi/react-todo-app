## Simple Click Todo

**GitHub Pages URL**: `https://shuunnichi.github.io/react-todo-app/`

### アプリケーション概要

「Simple Click Todo」は、ほとんどの操作がクリックのみでストレスなく操作できることを目指して開発されたTodo管理アプリケーションです。一般的なタスク管理機能に加え、**操作の取り消し・やり直し機能（Undo/Redo）**や、タスク入力を補助する独自機能に力を入れました。


### 主な機能

1.  **基本的なタスク管理**
    *   タスクの作成、名前の変更、完了設定ができます。
    *   タスクを完了すると完了日時が記録され、リスト内で自動でソートされます。
    *   起動時に、昨日以前に完了した古いタスクは**自動的に整理・削除されます**。

2.  **クイックピン機能（定型句の再利用）**
    *   よく使うタスク名やフレーズを「ピン」として登録し、ワンクリックで入力欄に呼び出せます。
    *   ピンが多くなりすぎた場合、画面が崩れないように自動で整理され、「もっと見る」ボタンで展開できます。

3.  **期限設定と視覚的な警告**
    *   週単位のカレンダービューで、今週や来週の日付を簡単に選択して期限を設定できます。
    *   だいぶ先の予定も、数値が入力できるので簡単に設定できます。
    *   タスクリストでは、**期限が過ぎたタスク**や**期限が近いタスク**に対し、分かりやすい色をつけて注意を促します。


4.  **安全なタスク削除**
    *   タスクを**右クリック**すると、すぐに削除されず「削除保留状態」に入ります。これは誤って重要なタスクを消してしまうのを防ぐ独自の仕組みです。



### 特に挑戦・工夫した点（オリジナリティ）

このアプリでは、使いやすさを高めるために、以下の工夫をしました。

1.  **操作をさかのぼる「元に戻す（Undo/Redo）」機能**
    *   アプリで行った**最新約50回分の操作（タスクの追加、変更、ピンの登録など）**をすべて記憶しています。
    *   これにより、ユーザーはキーボードショートカット（Ctrl+Z / Ctrl+Y など）を使って、いつでも操作前の状態に**完全に戻ったり**、**やり直したり**することができます。
    *   この機能を実現するために、タスクの状態が変化するたびに、システムが**タスクリスト全体をそっくり新しいものに作り直して**（スナップショットを取って）履歴として保存しています。

2.  **画面の大きさに合わせた定型句（ピン）の自動整理**
    *   タスク入力の補助となる「ピン」が多くなった場合でも、画面のレイアウトが崩れないよう、表示エリアをチェックしています。
    *   決まった行数（3行分）を超えてしまったピンは自動的に隠し、「もっと見る」ボタンで必要な時だけ表示・展開できるように制御しています。これは、画面の描画が終わった後でピンの位置や高さを正確に計測する**特殊な仕組み**を使って実現しています。

3.  **誤操作を防ぐための「削除保留システム」**
    *   一般的なTodoアプリでは、削除ボタンを押すと即座にタスクが消えてしまいますが、本アプリでは**右クリック**を起点とする独自の削除フローを導入しました。
    *   右クリックするとタスクが「削除を待っている状態」になり、誤操作を防ぎます。削除を確定するには再度右クリックします。

4. **安全な新規タスクの入力チェック（バリデーション）**
    *   入力に問題があるときは、**入力ボックスの枠を赤く変え**、**具体的な警告メッセージ**と**注意アイコン**を表示することで、ユーザーが何が間違っているかをすぐに理解できるように工夫しています。
